# web上机考试系统

#### 介绍
谢谦老师的创新实践项目——web上机考试系统

主要功能：

    - 管理增删改查老师、重置密码
    - 管理员查看所有考试的分数段分布、及格率、成绩统计图
    - 管理员修改全局配置信息
    - 管理员监测系统硬件配置
    
    - 老师增删改查考试
    - 老师导入学生名单、手动添加考生到一场考试
    - 老师查看某场考试具体情况、发送通知、下载答卷、打包班级答卷、评分
    - 老师查看自己创建的考试的分数段分布，及格率，成绩统计图
    
    - 学生查看自己参加的考试、下载试卷、上传答卷
    - 学生查看自己的各科分数、分数统计图
    - 学生个人云存储空间

    - 管理员、老师一键身份切换
    - 所有搜索均支持模糊查询
    - 数据库分页查询技术
